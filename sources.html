Public safety

The site can be imaginative and still be careful.

Cosmic Nexus is allowed to host speculative fiction, mythology, UAP scenarios, blue-domain hypotheses, travel quests and worldbuilding festivals. It should mark the difference between public fact, source trail, hypothesis, art, private memory and governance simulation every time the stakes rise.

Public factNeeds a visible source, date, author or repository trail.
HypothesisNeeds uncertainty, test path, disproof path and a calm public label.
AdventureCan be wild, funny and cinematic, as long as it does not pretend to be evidence or public permission.
